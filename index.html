snakemake cheatsheet
minimal rule
rule align_reads: input: r1 = "data/{sample}_R1.fastq.gz", r2 = "data/{sample}_R2.fastq.gz", ref = "ref/genome.fa" output: bam = "results/{sample}.bam" threads: 8 shell: """ bwa mem -t {threads} {input.ref} {input.r1} {input.r2} \ | samtools sort -o {output.bam} samtools index {output.bam} """
dry run & useful flags
# dry run — see what would execute snakemake -n --forceall # run on SLURM cluster snakemake --executor slurm --jobs 50 \ --default-resources slurm_partition="compute" mem_mb=8000 # generate rulegraph snakemake --rulegraph | dot -Tpng -o dag.png
💡 TIP
Always use --sdm conda or a Nix shell to pin all tool versions. Reproducibility is non-negotiable in bioinformatics.
